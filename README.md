# URBYS 🏙️🍃

**URBYS** é uma plataforma digital colaborativa desenvolvida para facilitar a comunicação entre a população e os responsáveis pela gestão urbana. O foco principal é o monitoramento e a preservação do meio ambiente nos centros urbanos.

Este projeto foi desenvolvido como um **MVP (Minimum Viable Product)** para validar a usabilidade e o engajamento cidadão no reporte de problemas ambientais, como descarte irregular de resíduos, poluição e queimadas.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias modernas de front-end para garantir uma experiência de usuário fluida e interativa:

* **Design:** [Figma](https://www.figma.com/) (Prototipação UI/UX)
* **Editor de Código:** [VS Code](https://code.visualstudio.com/)
* **Estrutura:** HTML5
* **Estilização:** CSS3 (Layout Responsivo)
* **Lógica:** JavaScript (ES6+)
* **Animações:** [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)

## 📋 Funcionalidades (MVP)

- [x] Interface intuitiva para reporte de problemas ambientais.
- [x] Formulário acessível para envio de denúncias.
- [x] Navegação dinâmica com animações GSAP.
- [x] Design focado em transparência e engajamento social.
